# Code of Conduct

## Our Pledge

We want this project to be a welcoming, respectful, and constructive place for everyone.

Participants are expected to:

- be respectful
- assume good intent
- give actionable technical feedback
- disagree without personal attacks
- focus on improving the project

## Expected Behavior

- Be polite and professional.
- Keep discussions technical and constructive.
- Accept that people have different levels of experience.
- Help make reviews and issue discussions useful for others.

## Unacceptable Behavior

- Harassment or personal attacks
- Insults, hostility, or trolling
- Discriminatory language or behavior
- Publishing private information without permission
- Repeated bad-faith disruption of discussions

## Scope

This applies in:

- issues
- pull requests
- discussions
- code review conversations

## Enforcement

Project maintainers may remove, edit, or reject comments, issues, pull requests, or contributions that do not align with this code of conduct.

Repeated or severe violations may lead to temporary or permanent exclusion from the project spaces.

## Reporting

If you need to report a problem, contact the project maintainer through the repository contact channel or a private communication path you trust.
